# Yggdrasil Saga

## Portrait Guidelines

**Version:** 5.0 — Canon Edition

**Authority:** CANON_REFERENCE.md

---

# Purpose

This document defines the standards used for every character portrait in **Yggdrasil Saga**.

It serves as the primary reference for:

- Character Portraits
- Dialogue Portraits
- Menu Portraits
- Emotion Sets
- UI Integration

Portraits should communicate personality before dialogue is read.

---

# Art Philosophy

Portraits should be:

- Clean
- Expressive
- Colourful
- Easy to read
- Faithful to Nintendo DS limitations

Avoid excessive rendering.

Strong silhouettes and facial expressions are preferred.

---

# Technical Standards

## Dialogue Portrait

Size

```text
128 × 128
```

Used during:

- Story dialogue
- Cutscenes
- Major quests

---

## Menu Portrait

Size

```text
64 × 64
```

Used for:

- Status screen
- Equipment
- Skills
- Save file

PNG

Transparent Background

# Standard Portrait Set

Every major character should include:

| Portrait   | Required |
|------------|----------|
| Neutral    | ✓        |
| Happy      | ✓        |
| Determined | ✓        |
| Surprised  | ✓        |
| Angry      | ✓        |
| Sad        | ✓        |
| Thinking   | ✓        |
| Serious    | ✓        |

Major story characters should also receive:

- Smiling
- Worried
- Shocked
- Emotional
- Closed Eyes

---

# Hero

Portraits should remain neutral enough for player projection.

Primary expressions:

- Neutral
- Determined
- Curious
- Concerned

Avoid exaggerated emotion.

---

# Ratatoskr

Ratatoskr is the most expressive character.

Required portraits:

- Neutral
- Happy
- Excited
- Worried
- Curious
- Thinking
- Shocked
- Sad
- Laughing

Tail position should change naturally between expressions.

---

# Astrid

Expressions

- Calm
- Determined
- Protective
- Smiling
- Angry
- Concerned

Astrid rarely appears frightened.

---

# Eirik

Expressions

- Curious
- Excited
- Thinking

- Happy

# Thora Ironhand

Expressions

- Neutral
- Focused
- Smirk

- Laughing

---

# Thrym

Expressions

- Calm
- Serious
- Proud
- Respectful

---

# Odin

Expressions

- Wise
- Calm
- Stern
- Reflective

Never appears panicked.

---

# Loki

Early Game

Expressions

- Friendly
- Relaxed
- Curious
- Amused

Late Game

Expressions

- Tired
- Honest
- Melancholic
- Resolute

Loki should never appear overtly villainous.

His tragedy should remain visible.

---

# Hel

Expressions

- Neutral
- Gentle
- Reflective
- Compassionate

Hel should always appear calm.

---

# Surtr

Expressions

- Proud
- Calm
- Determined

Never enraged.

Never monstrous.

---

# Dialogue Usage

Portrait changes should reinforce emotional shifts.

Avoid changing portraits every line.

Generally:

- New emotion
- New portrait

Not:

- Every sentence
- Every textbox

---

# UI Integration

Dialogue portraits appear on the:

Top Screen

Dialogue text appears on the:

Bottom Screen

Portraits should never overlap dialogue boxes.

---

# Accessibility

Expressions should remain readable at Nintendo DS resolution.

Facial features should remain exaggerated enough to communicate emotion clearly.

---

# Canon Principles

- Ratatoskr has the largest expression set.
- Loki's portraits evolve naturally throughout the story.
- Odin remains composed in every portrait.
- Hel is portrayed with compassion rather than fear.
- Portrait lighting reflects the realm currently being visited.
- Portraits support dialogue—they should never replace it.
