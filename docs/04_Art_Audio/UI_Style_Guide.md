# Yggdrasil Saga

## UI Style Guide

**Version:** 5.0 — Canon Edition

**Authority:** CANON_REFERENCE.md

---

# Purpose

This document defines the complete visual interface standard for **Yggdrasil Saga**.

It serves as the primary reference for:

- User Interface
- HUD
- Menus
- Battle Interface
- Touch Controls
- Dialogue Windows
- Icons
- Typography
- Accessibility

Every UI element should reinforce the game's Norse identity while remaining immediately readable on Nintendo DS
hardware.

---

# Core Philosophy

The interface should be:

- Clean
- Responsive
- Mythic
- Readable
- Minimal

The UI should never distract from gameplay.

---

# Visual Identity

Primary inspiration:

- Carved wood
- Ancient stone
- Rune carvings
- Norse knotwork
- Yggdrasil roots

Avoid futuristic or overly ornate elements.

# Touch Controls

Touch targets should never be smaller than:

```text
32 × 32
```

pixels.

Supports

- Tap
- Hold
- Drag
- Swipe

Entire game remains fully playable without touch input.

# Ratatoskr Interface

Ratatoskr manages:

- Fast Travel
- Journal
- Lore
- Codex Updates

His interface uses:

- Emerald highlights
- Root motifs
- Animated acorn icon

This visually distinguishes his menus from standard system menus.

---

# Canon Principles

- Dialogue portraits use **128×128** artwork.
- Ratatoskr manages all Fast Travel menus.
- The Rune Covenant has its own dedicated menu.
- UI styling reflects Norse craftsmanship rather than modern fantasy.
- Every interface element is designed for authentic Nintendo DS hardware.
- Readability always takes priority over decoration.
