# Purpose

The Bestiary records every enemy encountered throughout **Yggdrasil Saga**.

It serves as:

- Enemy encyclopedia
- Drop reference
- Weakness reference
- Lore archive
- Completion tracker

Every enemy is automatically recorded after its first defeat.

Additional information unlocks after repeated encounters.
