# Purpose

This document defines every equipment category used throughout **Yggdrasil Saga**.

It serves as the primary reference for:

- Equipment progression
- Weapon types
- Armour types
- Accessories
- Rune Equipment
- Upgrade paths
- Equipment balance

---

# Equipment Philosophy

Equipment should support:

- Character identity
- Class identity
- Realm progression
- Build diversity
- Long-term customization

Equipment should enhance strategy rather than replace it.
