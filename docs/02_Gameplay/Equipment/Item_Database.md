# Purpose

When conflicts occur, **CANON_REFERENCE.md** takes precedence.

# Item Categories

Items are divided into six categories.

| Category      | Purpose                    |
|---------------|----------------------------|
| Consumables   | Healing and battle support |
| Key Items     | Story progression          |
| Materials     | Crafting                   |
| Valuables     | Selling                    |
| Battle Items  | Combat only                |
| Special Items | Unique gameplay effects    |

# Canon Principles

- The Ten Covenant Runes are **not inventory items**.
- Rune Energy items enhance Rune abilities but never replace Covenant progression.
- Key Items are never sellable.
- Quest Items cannot be discarded.
- Arena Tokens are exclusive to Arena content.
- Mythic materials are reserved for postgame crafting.
