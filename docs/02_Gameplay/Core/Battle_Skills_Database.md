# Purpose

This document defines all combat skills available to:

* Wanderer
* Minstrel
* Rune Knight
* Guardian
* Seer
* Sage
* Valkyrie
* Berserker
* Runesmith
* Oracle
* Warden

All previous class skill databases are obsolete.

---

# Skill Elements

| Element  | Description                            |
|----------|----------------------------------------|
| Physical | Weapon-based damage                    |
| Rune     | Power drawn from the Ten Rune Covenant |
| Light    | Holy and radiant power                 |
| Fire     | Flame and destruction                  |
| Ice      | Frost and memory                       |
| Thunder  | Lightning and divine authority         |
| Nature   | Growth and life                        |
| Death    | Decay and mortality                    |
| Healing  | Restoration                            |
| Support  | Buffs and utility                      |

---

# Skill Target Types

| Type         | Description             |
|--------------|-------------------------|
| Single Enemy | One target              |
| All Enemies  | Entire enemy party      |
| Random Hits  | Multiple random targets |
| Self         | User only               |
| Ally         | One ally                |
| Party        | Entire party            |
