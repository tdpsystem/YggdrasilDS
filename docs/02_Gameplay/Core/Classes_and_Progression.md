# Yggdrasil Saga

## Classes and Progression

### Version 4.0 — Current Canon

---

# Purpose

This document defines the complete character progression system used throughout Yggdrasil Saga.

It serves as the primary reference for:

* Class progression
* Class changing
* Level progression
* Stat growth
* Skill acquisition
* Permanent abilities
* Class mastery
* Companion combat roles

This document supersedes all previous class documentation.

---

# Core Design Philosophy

The class system is designed around three principles:

1. Freedom of customization
2. Long-term progression
3. Distinct class identities

Players are encouraged to experiment with multiple classes while building unique party compositions.

# Class Change System

## Location

Class changing is performed at:

### Mímir's Well

Located beneath the roots of Yggdrasil.

---

## Narrative Role

Mímir's Well allows the Hero to attune their soul to different paths of knowledge and combat.

---

## Unlock Requirement

Complete the Prologue.

---

## Class Change Rules

When changing class:

* Character level resets to 1
* Learned permanent skills remain
* Equipment remains equipped if compatible
* Stats recalculate using the new class growth table
* Class mastery progress is retained

# Class Mastery

Each class possesses a Mastery Rank.

Mastery increases through:

* Battle participation
* Boss victories
* Arena challenges

---

## Mastery Tiers

| Rank        | Requirement    |
|-------------|----------------|
| Novice      | Class unlocked |
| Adept       | Level 20       |
| Expert      | Level 40       |
| Master      | Level 60       |
| Grandmaster | Level 99       |

* Fast Travel

# Endgame Progression

After unlocking all Advanced Classes:

Players may freely switch between every unlocked class at Mímir's Well.

Mastering multiple classes is expected for postgame content, Arena challenges, and New Game Plus.

* Freydis as party member
