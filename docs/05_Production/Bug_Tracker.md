# Known Risks

## High Risk

* Save corruption
* Event flag softlocks
* Party member recruitment bugs
* Ending route selection bugs

## Medium Risk

* Combat balancing
* Class progression pacing
* Dungeon encounter frequency

## Low Risk

* Text overflow
* Minor graphical issues
* Audio looping issues

---

# Release Criteria

Before Alpha:

* No critical bugs
* No progression blockers
* No save corruption

Before Beta:

* All high-priority bugs resolved
* Major balancing pass completed

Before Release:

* All critical bugs resolved
* All high-priority bugs resolved
* Remaining bugs documented and accepted
