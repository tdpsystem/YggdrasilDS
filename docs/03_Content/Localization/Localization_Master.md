# Purpose

This document defines the localization standards for **Yggdrasil Saga**.

It serves as the primary reference for:

- Translation
- Dialogue
- UI Text
- Menus
- Item Names
- Skill Names
- Quest Text
- Cultural Consistency

The goal is to ensure every localized version preserves the tone, mythology, and themes of the original game.

# Dialogue Rules

Dialogue should remain:

- Short
- Clear
- Easy to read

Nintendo DS dialogue boxes should generally remain under:

```text
120 characters
```

per text box whenever practical.

# Item Names

Item names should remain concise.

Maximum recommended length

```text
24 characters
```

---

# Skill Names

Maximum recommended length

```text
20 characters
```

Skill names should remain readable inside battle menus.

---

# Menu Text

Buttons

Prefer:

- Equip
- Use
- Drop
- Sort

Avoid unnecessarily long alternatives.

---

# Quest Titles

Maximum recommended length

```text
32 characters
```

Quest descriptions should prioritise clarity.

---

# UI Constraints

Nintendo DS resolution requires:

- Short labels
- Consistent spacing
- Predictable formatting

Text overflow must never occur.

---

# Number Formatting

Use locale-appropriate formatting where supported.

Examples

English

```text
9,999
```

Japanese

```text
9999
```

Currency displays should remain consistent.

---

# Text Encoding

Project Standard

UTF-8

Fallback

ASCII-compatible where required for engine compatibility.

---

# Voice Acting

The game does not use full voice acting.

Audio consists of:

- Battle shouts
- Reaction sounds
- UI voices
- Environmental voices

Localization should preserve timing for these effects.

Verify

- Text Overflow
- Grammar

- Canon Accuracy
- UI Fit
- Dialogue Timing

# Forbidden Changes

Localization must never alter:

- Story events
- Character motivations
- Rune names
- Realm order
- Ending structure
- Canon terminology

# Canon Principles

- English is the master script.
- CANON_REFERENCE.md is the highest authority.
- Character voices must remain consistent across languages.
- Mythological names should remain authentic whenever possible.
- Localization should preserve meaning rather than perform literal translation.
