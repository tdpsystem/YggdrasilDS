# Fast Travel Network

Ratatoskr unlocks travel between settlements as each realm is completed.

| Unlock       | Destination         |
|--------------|---------------------|
| Prologue     | Yggdrasil Sanctuary |
| Midgard      | Mannheim Village    |
| Alfheim      | Luminara            |
| Svartalfheim | Irondeep            |
| Vanaheim     | Verdant Grove       |
| Jötunheim    | Frosthold           |
| Niflheim     | Frozen Refuge       |
| Muspelheim   | Ember Bastion       |
| Asgard       | Valhalla            |
| Helheim      | Silent Crossing     |

Fast Travel is unavailable during:

- Active story events
- Final dungeon
- Certain boss encounters

# Canon Notes

## Permanent Companion

Ratatoskr accompanies the Hero throughout the entire adventure.

Functions:

- Fast Travel
- Journal
- Lore Commentary
- Quest Guidance
- Messenger between cosmic powers

Ratatoskr never fights and never takes sides.

## Canon Principles

- Every realm contains one primary settlement.
- Every settlement reflects the culture of its realm.
- Services expand naturally as the Hero progresses.
- Ratatoskr links every settlement through the Fast Travel network.
- Yggdrasil Sanctuary functions as both the beginning and ending hub of the game.
