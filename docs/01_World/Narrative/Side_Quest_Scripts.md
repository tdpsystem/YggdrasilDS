# Purpose

This document contains the narrative scripts for every major side quest in **Yggdrasil Saga**.

Unlike the Quest Database, this document focuses on:

- Dialogue
- Storytelling
- Character moments
- Worldbuilding
- Optional lore

Side quests should deepen the player's understanding of the Nine Realms without contradicting the main story.

---

# Script Format

Every side quest follows:

```text
Quest ID
Location
Quest Giver
Prerequisites
Objective
Dialogue
Completion
Rewards
Flags
```

# Canon Principles

- Side quests expand the world without altering the main story.
- Ratatoskr appears throughout every realm.
- Companion quests deepen character development.
- Side quests should reinforce the themes of memory, hope, renewal, and choice.
- No side quest may contradict the events of the main story.
