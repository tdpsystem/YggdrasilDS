# Purpose

This document defines every major non-playable character appearing throughout **Yggdrasil Saga**.

It serves as the primary reference for:

- Story NPCs
- Town NPCs
- Quest NPCs
- Shopkeepers
- Lore Characters
- Realm Leaders

When conflicts occur, **CANON_REFERENCE.md** takes precedence.

# Canon Notes

- Ratatoskr is the Hero's permanent companion.
- Loki is **not** the initial guide.
- Mímir's Well is the only class change location.
- Surtr is a story encounter rather than a traditional villain.
- Every realm contains NPCs reflecting its own philosophy toward the Rune Covenant.
- No deprecated NPCs appear in this document.
